Información¶

La aplicación Información de Odoo es una aplicación de productividad multiuso que permite a los usuarios internos enriquecer su base de conocimiento empresarial y proporcionar información recopilada de forma individual o grupal.

Las páginas en las que se recopila contenido se denominan artículos. Se componen principalmente de un título y un cuerpo. Este último es un campo HTML que contiene texto, imágenes, enlaces a otros artículos, registros de otros modelos, plantillas, etc.
