Inventario¶

La aplicación Inventario de Odoo es tanto una aplicación de inventario como un sistema de gestión de inventario. La aplicación permite a los usuarios gestionar los tiempos de espera, automatizar el reabastecimiento, configurar rutas avanzadas y mucho más.
