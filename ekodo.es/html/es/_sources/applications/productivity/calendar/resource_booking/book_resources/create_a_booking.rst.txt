================
Create a Booking
================

To create a booking, follow these steps:

#. Navigate to the :menuselection:`Resource Booking` module.
#. Click on :guilabel:`Resources`
#. In the calendar view, click on the desired date and time slot where you want to create a booking.
#. A form will appear allowing you to fill in the booking details.

 .. image:: create_a_booking/crear_reserva.png
      :alt: Booking Creation
      :align: center

#. Select the :guilabel:`Booking Type` from the list. This determines the type of resource combination that will be used.
#. Fill in the details of the booking, such as :guilabel:`Start Date`, :guilabel:`Duration`, and :guilabel:`Requester` (i.e., the person who is requesting the booking).

   .. image:: create_a_booking/ficha_reserva.png
      :alt: Create a book
      :align: center

#. Click :guilabel:`Save` to create the booking.
#. Optionally, click :guilabel:`Confirm` to finalize the booking if all details are correct.
